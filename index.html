Im a Computer Science student with an interest in Graphics, Unity, Quantum Computing, Software engineering, and anything Programming.

The GEMS project has been a fun summer project for me, focusing on shader development, webGL in Javascript, Three.Js, and procedural development. A custom shader is mapped onto a random 3D object

In a brief explanation. The input value is converted to a number in base-36. Then many of its properties are tested and matched with attributes of the shader. For example, the remainder of the number divided by 3 will determine one of the textures used in the object, or the most common digit being used to determine the amount of colors in the object.

To generate some objects, try typing a value in the text box labeled seed. Some of my favorite seed values have been Gems, Plox, 1234567890, Spring, 6o4sfs
